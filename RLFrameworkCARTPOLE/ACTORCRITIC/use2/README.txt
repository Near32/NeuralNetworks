run ./configure ./make ./runDECEMBER and you are good to go with 32 threads working concurrently to optimize the Cart Pole Problem. You can see the progress by reading the files in the DATAS directory, for each episode corresponds a file. In columns, you have : x theta xdot thetadot Action. Where (x,theta) should remain at (0,pi) and Action is the force that is applied by the agent at each time step...
